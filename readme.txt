== Description ==

= Ng Gallery Optimizer Modified =
It is a modified version of [Nextgen Gallery optimizer plugin](http://wordpress.org/extend/plugins/nextgen-gallery-optimizer/) ,which only gives support for "nggallery" shortcode for free.

It improves your site's page load speed by ensuring NextGEN Gallery's scripts and styles ONLY load on posts with the nextgen galley shortcode.

It also includes and *automatically*-integrates the fantastic [Fancybox](http://fancybox.net) lightbox script, so now you can have gorgeous galleries AND a speedy site! *Requires [NextGEN Gallery](http://wordpress.org/extend/plugins/nextgen-gallery) 1.6.2 and up.

Supports all ten shortcodes of Nextgen Gallery  [nggallery id=x], [slideshow id=x], [album id=x], [thumb id=x], [singlepic id=x], [imagebrowser id=x], [nggtags gallery|album=mytag], [random max=x], [recent max=x] and [tagcloud].

It also adds support for the [Show as slideshow] link (loading slideshow scripts only after a user clicks-through)


== Installation ==

= Install Ng Gallery Optimizer Modified: =

1. Download Ng Gallery Optimizer Modified to your computer
2. Log into your WordPress admin
3. Click **Plugins --> Add New**
4. Click **Upload** (in the header)
5. Search your computer for **ng-gallery-optimizer-modified.zip**
6. Click **Install Now**
7. Click **Activate Plugin**
8. Click **Gallery --> Ngg Optimizer Modified** to complete installation.

* Note: If your computer automatically unzips your download, you'll need to re-compress it first.
* On Mac OSX: Ctrl+click the plugin folder --> Compress.
* On Windows: Right-click the plugin folder --> Send To --> Compressed (zipped) Folder.
